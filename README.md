# EV Analytics Dashboard

### Live Demo: https://analytics-dashboard-assessment-weld-two.vercel.app/

A responsive analytics dashboard visualizing Electric Vehicle population data.

### Features
* **Dynamic Filtering:** Filter data by County.
* **Interactive Charts:** Adoption trends, Market share (BEV vs PHEV), and Range analysis.
* **Data Grid:** Paginated table to explore raw registration data.
* **Performance:** Optimized with React `useMemo` for efficient data processing.

### Tech Stack
* React + Vite
* Tailwind CSS
* Recharts
* PapaParse
